OutputPanel

Output panel is a container element with several use cases such as autoUpdate and dynamic loading. This example demonstrates the dynamic loading feature where the content of the panel is loaded after the page itself is loaded to speed up initial page load time. A progress indicator is displayed while loading the content.

This part of the page is loaded lazily after initial page load.
