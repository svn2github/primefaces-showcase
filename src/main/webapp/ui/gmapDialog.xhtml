GMap - Dialog

Special care needs to be taken when gmap is place inside containers that are initially hidden. In this example, dialog's onShow event is use to calculate the actual dimensions of map.

Source

<p:commandButton type="button" image="ui-icon ui-icon-pin-s"
        value="Show Map" onclick="dlg.show()" />

<p:dialog widgetVar="dlg" width="625" height="450" modal="true"
    onShow="mymap.checkResize()">
    <p:gmap center="41.381542, 2.122893" zoom="15" type="HYBRID" style="width:600px;height:400px" widgetVar="mymap"/>
</p:dialog>
